## Getting Started

This example demonstrates how to use the NodeJS SDK with Next.js (App Router) and [Middleware](https://nextjs.org/docs/14/app/building-your-application/routing/middleware).

## How to run

- Update your site code in the file: `src/kameleoon.ts`.
- Update your feature key in the file: `src/app/api/route.ts`.
- Run `yarn ..` or `npm ...`, then open your browser and go to `http://localhost:3000`.
- To simulate a new visitor please remove the `kameleoonVisitorCode` cookie in your browser.

For Yarn:

```bash
yarn install
```
development:

```bash
yarn dev
```
production:

```bash
yarn build
yarn start
```

Or for NPM:

```bash
npm install
```
For development:

```bash
npm run dev
```
For production:

```bash
npm run build
npm run start
```
